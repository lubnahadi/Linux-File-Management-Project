# Linux-File-Management-Project

## Project Overview 
This project demonstrates file and directory management in a Linux envoirement using command-linetools.It focuseson navigating the file system, creating and managing directories, and understanding basic Linux operations.

## Skills Demonstrated
-Linux command line usage
-File and directory management
-Navigation using terminal commands
-Basic troubleshooting
-System interaction

## Tools Used
-Linux (Ubuntu/Fedora)
-Bash Shell
-Terminal

## Tasks Performed
-Created and managed directories using commands like `mkdir` and `rmdir`
-Navigated the file system using `cd`, `ls` , and `pwd`
-Managed files and folders in a Linux envoirement
-Practiced command-line opeartions for system tasks

## What i learned
This project helped me understand how to work with the Linux file systemand use command-line tools efficiently.These skills are important for IT support and entry level system administration roles.
